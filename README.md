# README.md

## Overview

This project is a **Python code generator for a very simple register/cell machine**.
Its version v0.0.1.
Instead of performing arithmetic directly in Python, the functions generate a **string containing assembly-like instructions** for the Bonsai assembler environment:

https://inf-schule.de/rechner/bonsai/sprachen/assemblersprache/experimente

Each function produces instructions such as:

* `inc`
* `dec`
* `tst`
* `jmp`
* `hlt`

These instructions operate on **cells (memory locations)** and are concatenated to form a complete program. The generated program is then printed and can be executed in the Bonsai assembler simulator.

---

# Machine Instruction Set

The generated programs use a very small instruction set.

---

## `inc <cell>`

Increments the value stored in a cell by **1**.

Example:

```
inc 3
```

If **cell 3 = 2**, it becomes:

```
cell 3 = 3
```

---

## `dec <cell>`

Decrements the value stored in a cell by **1**.

Example:

```
dec 3
```

If **cell 3 = 3**, it becomes:

```
cell 3 = 2
```

---

## `tst <cell>`

Tests whether a cell contains **0**.

The instruction following `tst` determines the next step:

* If the cell **is not zero**, execution continues with the **next instruction**.
* If the cell **is zero**, execution skips the next instruction and continues with the **instruction after that**.

Example:

```
tst 3
inc 2
dec 2
```

If **cell 3 ≠ 0**, then:

```
inc 2
```

is executed.

If **cell 3 = 0**, the `inc` is skipped and execution continues with:

```
dec 2
```

---

## `jmp <address>`

Unconditional jump to a specific **program line number**.

Example:

```
jmp 42
```

Execution continues at instruction **42**.

---

## `hlt`

Stops program execution.

---

# Operations Generated by Python

The Python functions generate instruction sequences that implement higher-level operations.

---

# `save(location, data)`

## Purpose

Stores a **constant value** in a cell.

## Behavior

1. The cell is first reset to `0`.
2. The cell is incremented `data` times.

## Result

```
location = data
```

Example:

```python
save(7, 5)
```

Result:

```
cell7 = 5
```

---

# `null(location)`

## Purpose

Sets a cell to **0** regardless of its previous value.

## Behavior

The program repeatedly decrements the cell until it reaches zero.

## Result

```
location = 0
```

Example:

```python
null(3)
```

If:

```
cell3 = 7
```

After execution:

```
cell3 = 0
```

---

# `add(a, b, result)`

## Purpose

Adds the values of two cells.

## Behavior

The algorithm transfers units from the input cells into the result cell:

1. Decrement `a` and increment `result` until `a` becomes 0.
2. Decrement `b` and increment `result` until `b` becomes 0.

## Result

```
result = a + b
```

## Side Effects

The input cells are consumed:

```
a = 0
b = 0
```

Example:

```
cellA = 3
cellB = 2
```

After execution:

```
result = 5
cellA = 0
cellB = 0
```

---

# `sub(a, b, result)`

## Purpose

Computes the **difference between two cells**.

## Behavior

1. Each unit of `a` increments `result`.
2. Each unit of `b` decrements `result`.

## Result

```
result = a - b
```

## Side Effects

```
a = 0
b = 0
```

Example:

```
cellA = 5
cellB = 3
```

After execution:

```
result = 2
```

# Summary

This project implements a **macro assembler generator in Python** for a minimal computational model based on:

* increment
* decrement
* conditional tests
* jumps

From these primitive operations, more complex operations such as:

* constant assignment
* clearing cells
* addition
* subtraction

can be built.
